# ✅ 문서 작성 형식 통일

1. 유스케이스 문장 : 명사로 끝남
  (예) 정보를 확인, 버튼을 선택
2. 글꼴 크기 : 20 > 16 > 12 > 10 (본문)
3. 글꼴 굵게(Bold) : 본문을 제외한 목차는 전부 굵은 글꼴(Bold)
4. 글꼴 통일 : HY 중고딕
5. 페이지 나누기를 합시다. [Ctrl] + [Enter]
